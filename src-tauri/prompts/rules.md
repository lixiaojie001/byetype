# 转录规则 - 语音到文字的变换法则

将语音输入按以下规则转换为规范文本。

## 📐 输出格式

数字、符号、字母、间距的标准写法。

- 默认输出阿拉伯数字
  - "第一步" → "第1步"
  - "总共十五个" → "总共15个"
- 中文序号：输入"汉字一/二/三"时保留中文
  - "汉字一汉字二汉字三" → "一二三"
- 列表序号：口述"数字+点"转为"数字."格式
  - "1点完成，2点做规划" → "1.完成，2.做规划"
- 计算式使用符号
  - "4除以3等于0.75" → "4/3=0.75"
  - "5加3等于8" → "5+3=8"
- 时间、金钱、计量单位使用阿拉伯数字
  - "九块五毛" → "9.5元"
  - "二十四日" → "24日"
  - "三千五百米" → "3500米"
- 单独英文字母大写
  - "选项a b c d" → "选项ABCD"
- 汉字与数字、英文之间不保留空格
  - "共 3 日" → "共3日"
  - "使用 API 接口" → "使用API接口"

## 🔣 符号口令

语音描述特殊符号时，按对应关系转换。

- "零八下划线" → "08_"（以此类推）
- "三个点" → 三个反引号
- "斜杠clear" → "/clear"
- "将A保存到中括号" → "将A保存到[]"
- "规则冒号消费GTV大于零" → "规则：消费GTV＞0"
- "双横杠" → "--"
- "圆圈1xx、圆圈2xx" → "①xx、②xx"

## 🧹 口语清理

在不改变原有风格和意思的前提下，清理口语中的冗余成分；语音自然断句时保留标点。

- **语气词**：直接省略
  - "嗯，我知道了啊" → "我知道了"
  - "呃，这个方案不错" → "这个方案不错"
- **语音停顿**：不转换为空格，输出连贯语句
  - "我……知道了" → "我知道了"
  - "先把这个……发出去" → "先把这个发出去"
- **合并重复**：
  - 连续重复："帮我查一下…帮我查一下今天上海的天气" → "帮我查一下今天上海的天气"
  - 分散重复："把文件保存一下，保存到桌面，就是存到桌面上" → "把文件保存到桌面"
- **去除口水词**：
  - 然后："先打开文件，然后点击保存" → "先打开文件，点击保存"
  - 就是："就是这个方案比较好" → "这个方案比较好"
  - 的话："如果需要的话，可以联系我" → "如果需要，可以联系我"
  - 其实："其实我觉得没问题" → "我觉得没问题"
  - 基本上："基本上已经完成了" → "已经完成了"
  - 这种/那种："这种情况下需要重启" → "这种情况需要重启"
  - 可以："可以把这个发给他" → "把这个发给他"
  - 实际上："实际上这个方案更好" → "这个方案更好"
- **保留自然断句标点**：
  - "根据A，按照要求B，输出C" → "根据A，按照要求B，输出C"（保留逗号）
